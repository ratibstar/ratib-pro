# Accounting System - Quick Start Guide

## 5-Minute Quick Start

### Step 1: Access the System (30 seconds)
1. Login to your account
2. Click "Accounting" in the main menu
3. You're in!

### Step 2: Check Your Dashboard (1 minute)
- View summary cards (Revenue, Expenses, Cash, Bank)
- Click "View Transactions" to see all transactions
- Familiarize yourself with the layout

### Step 3: Create Your First Journal Entry (2 minutes)
1. Click "New Journal Entry"
2. Enter:
   - Date: Today
   - Description: "Test Entry"
   - Account: Select "Cash"
   - Debit Amount: 100
   - Currency: SAR
3. To balance, add:
   - Account: Select "Owner Capital"
   - Credit Amount: 100
4. Check: Balance Status should be GREEN ✅
5. Click "Save"

### Step 4: View Your Entry (30 seconds)
- Go to "Journal Entries" tab
- Find your entry in the table
- Click "View" to see details

### Step 5: Explore Other Features (1 minute)
- Try "Chart of Accounts" - see all accounts
- Try "Vouchers" - create a test payment voucher
- Try "Banking & Cash" - add a bank account
- Try "Financial Reports" - generate a Trial Balance

## Common Tasks - Quick Reference

### Recording a Payment
1. Go to "Vouchers" → "Payment Voucher"
2. Fill: Date, Method, Amount, Party
3. Save

### Recording a Receipt
1. Go to "Vouchers" → "Receipt Voucher"
2. Fill: Date, Method, Amount, Party
3. Save

### Creating a Bank Transaction
1. Go to "Banking & Cash" → "New Transaction"
2. Select bank, date, type, amount
3. Save

### Reconciling Bank Account
1. Go to "Banking & Cash" → "Reconcile"
2. Select account, enter statement date and balance
3. Review difference (if any)
4. Start reconciliation

### Generating a Report
1. Go to "Financial Reports"
2. Click on any report card
3. Review the report
4. Print if needed

## Essential Rules

1. **Debits MUST equal Credits** in every journal entry
2. **Always check the balance status** (should be green)
3. **Use clear descriptions** for easy reference
4. **Reconcile monthly** to catch errors early
5. **Link to entities** when applicable
