## 🚀 Getting Started

### Step 1: Logging In

1. **Open your web browser** (Chrome, Firefox, or Edge recommended)
2. **Navigate to** your Ratib Program URL (provided by your administrator)
3. **You will see the login page** with:
   - Username field (top)
   - Password field (below username)
   - "Login" button
   - "Forgot Password?" link (if available)

4. **Enter your credentials:**
   - Type your **username** in the first field
   - Type your **password** in the second field
   - Click the **"Login"** button

5. **After successful login**, you will be automatically redirected to the **Dashboard**

> 💡 **Tip:** If you forget your password, click "Forgot Password?" and follow the instructions sent to your email.

---

When you first log in, you'll see the **Dashboard**. This is your main control center where you can see:
- **Overview statistics** for all modules

## 🔧 Workers Management

### What is Workers Management?

The Workers module lets you manage all worker profiles, documents, and statuses.

### Accessing Workers

1. Click **"Workers"** in the left menu (🔧 icon)
2. Or click the **Workers card** on the Dashboard

### Understanding the Workers Page

When you open Workers, you'll see:

#### Top Section - Search and Filters

- **Search Box** - Type to search by name, passport number, or nationality
- **Status Filter** - Filter by status (All, Pending, Approved, Deployed, etc.)
- **Agent Filter** - Filter by agent
- **Add Worker Button** - Create a new worker profile

#### Main Table

The table shows all workers with columns:
- **Name** - Worker's full name
- **Passport Number** - Passport ID
- **Nationality** - Country
- **Agent** - Assigned agent
- **Status** - Current status (color-coded)
- **Actions** - Buttons to view, edit, or delete

### Adding a New Worker

1. **Click the "Add Worker" button** (usually green, top right)
2. **Fill in the form:**
   - **Personal Information:**
     - Full Name (required)
     - Passport Number (required)
     - Date of Birth
     - Gender
     - Nationality
     - Contact Number
     - Email Address
     - Address
   
   - **Work Assignment:**
     - Select Agent (from dropdown)
     - Select Subagent (optional)
     - Visa Type
     - Job Category
     - Salary
   
   - **Status:**
     - Select initial status (usually "Pending")
   
3. **Click "Save" or "Submit"** button
4. **Success message** will appear - worker is now added!

### Editing a Worker

1. **Find the worker** in the table (use search if needed)
2. **Click the "Edit" button** (pencil icon ✏️) in the Actions column
3. **Modify the information** you need to change
4. **Click "Update" or "Save"** button
5. **Changes are saved** - you'll see a confirmation message

### Viewing Worker Details

1. **Click the "View" button** (eye icon 👁️) in the Actions column
2. **A detailed view opens** showing:
   - All personal information
   - Documents (if uploaded)
   - Status history
   - Related information

### Uploading Worker Documents

1. **Open a worker's profile** (click View or Edit)
2. **Find the "Documents" section**
3. **Click "Upload Document"** button
4. **Select document type:**
   - Identity Document
   - Passport
   - Medical Certificate
   - Police Clearance
   - Visa
   - Ticket
5. **Choose file** from your computer
6. **Click "Upload"**
7. **Document appears** in the documents list

### Changing Worker Status

1. **Find the worker** in the table
2. **Click the status dropdown** (or Edit button)
3. **Select new status:**
   - **Pending** - Initial status, awaiting approval
   - **Approved** - Approved and ready
   - **Rejected** - Application rejected
   - **Deployed** - Worker is deployed
   - **Returned** - Worker has returned
4. **Click "Update Status"**
5. **Status changes** - color updates automatically

### Bulk Operations

You can perform actions on multiple workers:

1. **Select workers** using checkboxes (left column)
2. **Choose action** from bulk actions dropdown:
   - Bulk Activate
   - Bulk Deactivate
   - Bulk Delete
   - Bulk Status Update
3. **Click "Apply"**
4. **Confirm** the action
5. **All selected workers** are updated

### Searching Workers

1. **Type in the search box** (top of table)
2. **Search by:**
   - Name
   - Passport number
   - Nationality
   - Agent name
3. **Results filter automatically** as you type
4. **Clear search** by deleting text

## 💰 Accounting System

### What is the Accounting System?

The Accounting module is a complete financial management system for tracking money, transactions, invoices, bills, and generating financial reports.

### Accessing Accounting

1. Click **"Accounting"** in the left menu (💰 icon)
2. Or click the **Accounting card** on the Dashboard

### Understanding Accounting Tabs

When you open Accounting, you'll see tabs at the top:

#### 1. Control Panel (Dashboard)
- **Overview** of financial status
- **Quick statistics:**
  - Total Invoices
  - Total Bills
  - Bank Balances
  - Accounts Receivable
  - Accounts Payable
- **Charts** showing financial trends

#### 2. Chart of Accounts
- **List of all accounts** in your system
- **Account categories:**
  - Assets (what you own)
  - Liabilities (what you owe)
  - Equity (your investment)
  - Income (money coming in)
  - Expenses (money going out)

**How to Use:**
1. **View accounts** in a tree structure
2. **Click account name** to see details
3. **Add new account** using "Add Account" button
4. **Edit account** by clicking edit icon

#### 3. Journal Entries
- **Record manual transactions**
- **Create entries** with debits and credits

**Creating a Journal Entry:**
1. **Click "New Journal Entry"** button
2. **Enter date** and reference number
3. **Add line items:**
   - Select Account
   - Enter Debit amount OR Credit amount
   - Add description
4. **Ensure debits = credits** (system will warn if not balanced)
5. **Click "Save"**

#### 4. Invoices (Receivables)
- **Create invoices** for money customers owe you
- **Track payments** received

**Creating an Invoice:**
1. **Click "New Invoice"** button
2. **Fill in:**
   - Customer Name
   - Invoice Date
   - Due Date
   - Items/Services
   - Amount
3. **Click "Save"**
4. **Invoice is created** - appears in receivables

#### 5. Bills (Payables)
- **Record bills** you need to pay
- **Track payments** made

**Creating a Bill:**
1. **Click "New Bill"** button
2. **Fill in:**
   - Vendor Name
   - Bill Date
   - Due Date
   - Items/Services
   - Amount
3. **Click "Save"**
4. **Bill is created** - appears in payables

#### 6. Banking & Cash
- **Manage bank accounts**
- **Record bank transactions**
- **Reconcile bank statements**

**Adding a Bank Account:**
1. **Click "Add Bank Account"** button
2. **Enter:**
   - Bank Name
   - Account Number
   - Account Type
   - Opening Balance
3. **Click "Save"**

**Recording a Bank Transaction:**
1. **Select bank account**
2. **Click "New Transaction"**
3. **Enter:**
   - Date
   - Type (Deposit/Withdrawal)
   - Amount
   - Description
4. **Click "Save"**

#### 7. Financial Reports
- **Generate various reports:**
  - Trial Balance
  - Balance Sheet
  - Income Statement
  - Cash Flow Statement
  - Profit & Loss
  - And many more...

**Generating a Report:**
1. **Click "Financial Reports" tab**
2. **Select report type** from dropdown
3. **Choose date range:**
   - From Date
   - To Date
4. **Click "Generate Report"**
5. **Report displays** - you can print or export

### Understanding Accounting Colors

- 🟢 **Green** - Positive balances, income
- 🔴 **Red** - Negative balances, expenses
- 🔵 **Blue** - Information, neutral

## 🔧 Troubleshooting for Users

### Common Issues and Solutions

#### Problem: Can't Log In

**Possible Causes:**
- Wrong username or password
- Account is disabled
- Browser issues

**Solutions:**
1. **Check username and password** - Make sure Caps Lock is off
2. **Click "Forgot Password"** - Reset your password
3. **Clear browser cache** - Clear cookies and cache
4. **Try different browser** - Use Chrome or Firefox
5. **Contact administrator** - If nothing works

#### Problem: Can't See a Module

**Possible Causes:**
- You don't have permission
- Module is disabled

**Solutions:**
1. **Check with administrator** - Ask for permission
2. **Refresh page** - Sometimes permissions update after refresh
3. **Logout and login again** - Refresh your session

#### Problem: Page Won't Load

**Possible Causes:**
- Internet connection issue
- Server problem
- Browser issue

**Solutions:**
1. **Check internet connection** - Make sure you're online
2. **Refresh page** - Press F5 or click refresh button
3. **Try different browser** - Switch browsers
4. **Clear browser cache** - Clear cache and cookies
5. **Contact IT support** - If problem persists

#### Problem: Can't Save Data

**Possible Causes:**
- Required fields missing
- Validation errors
- Server issue

**Solutions:**
1. **Check for red error messages** - Read error messages carefully
2. **Fill required fields** - Fields with * are required
3. **Check data format** - Dates, numbers must be in correct format
4. **Try again** - Sometimes temporary server issues
5. **Contact support** - If problem continues

#### Problem: Search Not Working

**Possible Causes:**
- Typed incorrectly
- No matching results
- Search index issue

**Solutions:**
1. **Check spelling** - Make sure words are spelled correctly
2. **Try different keywords** - Use alternative search terms
3. **Clear search** - Clear and try again
4. **Use filters** - Try using filters instead

#### Problem: Can't Upload Document

**Possible Causes:**
- File too large
- Wrong file type
- Browser issue

**Solutions:**
1. **Check file size** - Make sure file is under size limit
2. **Check file type** - Use allowed formats (PDF, JPG, PNG)
3. **Try different file** - Test with another file
4. **Try different browser** - Switch browsers
5. **Contact support** - If nothing works

### Getting Help

If you encounter problems:

1. **Check Help Center** - Search for your issue
2. **Read Error Messages** - Error messages often tell you what's wrong
3. **Contact Administrator** - Reach out to your system administrator
4. **Report Bug** - If you find a bug, report it with details:
   - What you were trying to do
   - What happened instead
   - Error messages (if any)
   - Screenshots (if possible)
