# CoreAI

CoreAI is a fully isolated, clean-architecture project scaffold.

## Folder Layout

- `frontend/`: UI-facing assets and templates
- `backend/`: server-side application logic
- `css/`: global and modular styles
- `js/`: client-side scripts
- `php/`: shared PHP bootstrap and core helpers
- `ai/`: AI-specific services and integrations
- `context/`: context payloads and domain context data
- `agent/`: agent workflows and orchestration

## Entry Point

- `index.php`: main application entry point

## Isolation Rule

All CoreAI code and assets live inside `coreai/` only.
